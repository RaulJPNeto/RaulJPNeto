## Projetos em Destaque

### 🎫 [Support Ticket API](https://github.com/RaulJPNeto/support-ticket-api)

API REST para gerenciamento de tickets de suporte técnico, com arquitetura em camadas, controle de acesso por perfil e documentação interativa.

**Stack:** Laravel · PostgreSQL · Docker · Sanctum · Swagger

**Destaques:**
- Arquitetura em camadas: Controller → Service → Query → Resource
- Autenticação via token (Laravel Sanctum)
- Autorização por perfil: Client, Agent e Admin
- Soft delete com restore e force delete
- Histórico de mudanças de status por auditoria
- Events e notificações por email
- Documentação interativa com Swagger UI
- Testes automatizados com PHPUnit
- Análise estática com Larastan
